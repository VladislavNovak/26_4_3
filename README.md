<summary><span style="color:tomato;font-size:12px">Реализация программы управления окном рабочего стола</span></summary>

Реализуйте симуляцию работы окна в визуальном интерфейсе пользователя.

Окно располагается на виртуальном экране монитора, размером 80 X 50 точек. 
У окна есть несколько параметров, которые также задаются в пикселях: 
двумерные координаты левого верхнего угла и размеры, ширина и высота. 
Ширина и высота при этом не могут быть отрицательными, а координаты не могут выходить за область экрана.

Пользователь взаимодействует с программой с помощью команд:

- Команда move принимает вектор, на который надо сместить окно на экране. Сообщение с новыми координатами окна появляется на экране.
- Команда resize запрашивает из стандартного ввода новый размер окна. После изменения размера сообщение с новым размером выводится на экране.
- Команда print выводит текущее изображение монитора в консоль. 0 — это пиксель вне области окна, 1 — с окном.
- Команда close закрывает окно, выход из программы.

<summary><span style="color:tomato;font-size:12px">Реализация</span></summary>

Класс Window содержит информацию о типе. 
Это может быть простое окно SIMPLE, а может быть MONITOR. 

MONITOR содержит информацию лишь о максимально возможных размерах. 
По сути, это виртуальное представление монитора с дисплеем. 
Размер можно задать лишь раз - при инициализации

SIMPLE можно задавать размеры (ширину и высоту), но лишь в пределах, 
которые оно автоматически получает от ранее созданного монитора. 
У SIMPLE возможно изменение размеров 
(в классе это реализовано как changeWidth и changeHeight, но нужно добавить в меню).

Так же есть класс Screen. Он содержит vector<Window*> windows. 
Первым элементом в этом векторе всегда идёт window->MONITOR, который создаётся при инициализации Screen. 
Остальные элементы - это окна SIMPLE, которые создаются посредством Screen::addWindow().

Итогом работы является возможность добавления 
в объект screen любого количества окон с последующим доступом к любому из них.
